# "뉘앙스" 글리프 듀엣 — 그래픽 스코어 (Score 버전)

2인 타이핑 대화 게임 / 공연용 오디오비주얼 작품. 이 폴더는 **스코어 버전만** 담은
독립 실행본입니다. 그대로 다른 컴퓨터로 옮겨서 실행할 수 있습니다.

## 실행 방법

> ⚠️ 중요: HTML 파일을 더블클릭해서(`file://`) 열면 **작동하지 않습니다.**
> 브라우저가 보안상 ES 모듈(자바스크립트)을 차단하기 때문에, 반드시 **로컬 서버**로 열어야 합니다.

### 1) Python으로 (가장 간단 — 별도 설치 거의 없음)

macOS/Linux에는 Python3가 기본 설치되어 있습니다. 이 폴더에서 터미널을 열고:

```bash
python3 serve.py 8777
```

그다음 브라우저에서 **http://localhost:8777/** 접속 → **스코어 버전**이 바로 열립니다.
(이 저장소는 스코어 버전만 담겨 있어 `index.html`이 곧 스코어 버전 = 루트 `/` 가 스코어 버전입니다.)

(포트는 바꿔도 됩니다: `python3 serve.py 9000` → http://localhost:9000/)

Python의 기본 서버만 써도 됩니다:

```bash
python3 -m http.server 8777
```

### 2) Node가 있다면

```bash
npx serve .        # 또는  npx http-server -p 8777
```

## 필요한 것

- **로컬 HTTP 서버** (위의 Python 또는 Node 중 택1)
- **인터넷 연결** — 글꼴(Datatype, Galmuri)을 CDN에서 불러옵니다.
  완전 오프라인으로 쓰려면 글꼴을 로컬로 받아 `index.html`의 `<link>`를 교체하세요.
- 최신 크롬/엣지/사파리 권장. 첫 화면의 ▶ 버튼을 눌러야 소리가 켜집니다(브라우저 자동재생 정책).

## 조작

- ▶ 시작 → ▶ 대화 시작 → **뉘앙스 라운드 사이클**
- 라운드는 **32→16→8→4초로 순환 반복**, 매 라운드 앞에 **4초 예비박**(카운트인 4·3·2·1)
- 라운드 동안: **캐릭터 4개 중** 골라 타이핑(**Enter** 전송, **Tab** 차례 넘김) + 부호 투표 동시
- 퍼포머가 **■**(왼쪽 위)로 마치면 → **선물 단계**(드래그&드롭) → ▶ → 엔딩
- 엔딩: ① 순차 듀엣 악보 → ② 랜덤 입장 오케스트라 총보(합주). 마칠 때의 뉘앙스 말투가 전체 사운드에 입혀진 채 연주

## 관객 참여 — 뉘앙스 라운드 사이클

대화 타이핑이 시작되면 라운드가 **32→16→8→4초로 순환하며 계속 반복**된다(점점 짧아져
긴박해짐). 각 라운드 앞엔 **4초 예비박**(상단 가운데 4·3·2·1, 또각또각 카운트). 라운드
동안 퍼포머·관객이 6개 부호 `.` `?` `!` `…` `~` `;` 에 **투표**한다(예비박 중에도 투표 가능).

- 누를 때마다 **인스타 라이브 하트처럼 부호 아이콘이 떠오르고**, 버튼에 실시간 카운트가 쌓인다.
- **가장 많이 눌린 부호가 사운드 이펙트를 실시간으로 결정**한다(리더가 바뀌면 곧바로 반영).
  - `.` 담담·마른 / `?` 되묻듯·들뜬 / `!` 강조·날카로움 / `…` 머뭇·아득함 / `~` 물결·흔들림 / `;` 망설임·머무름
- 퍼포머가 **■**로 사이클을 마치면 **그때의 말투가 엔딩까지 고정**되고, 선물 단계로 넘어간다.

- **퍼포머**: 노트북에서 타이핑(키보드) + 부호 패드 클릭(마우스) + ■로 마치기. 관객 투표가 주로 폰에서 들어온다.
- **관객 폰**: 라운드 동안 뜨는 **QR**을 찍어 `tap.html`을 열면 실시간으로 함께 투표할 수 있다.

### 관객 폰을 같은 와이파이로 받기 (기본)
`serve.py`를 띄우면 콘솔에 `http://<노트북 LAN IP>:8777/tap.html` 주소가 찍힌다.
관객이 **같은 와이파이(또는 노트북 핫스팟)** 면 메인 화면의 QR로 바로 접속된다.

### 관객 폰을 **데이터(셀룰러)** 로도 받기 — 터널
같은 와이파이가 아니어도 받으려면, 로컬 서버를 터널로 잠깐 공개해 공개 https
주소를 만들고 그 주소를 QR에 넣으면 된다(노트북에 인터넷만 있으면 됨):

```bash
# 1) 서버 실행
python3 serve.py 8777
# 2) 다른 터미널에서 터널 (계정 불필요)
cloudflared tunnel --url http://localhost:8777
#   → https://xxxx.trycloudflare.com 같은 공개 주소가 나온다
# 3) 메인 화면을 그 공개 주소로 알려주며 연다
#   http://localhost:8777/?pub=https://xxxx.trycloudflare.com
```

이러면 QR이 공개 주소를 가리켜 **어떤 망(데이터/와이파이)** 에서도 접속된다.
터널을 안 쓰면 자동으로 같은-와이파이(LAN) 방식으로 폴백한다. (ngrok도 동일하게 가능.)
